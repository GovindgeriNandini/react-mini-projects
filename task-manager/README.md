#  Installation

Clone the repository

```
git clone https://github.com/GovindgeriNandini/react-mini-projects.git
```

Navigate to the project folder

```
cd react-mini-projects
```

Enter any project

```
cd event-handling
```

Install dependencies

```
npm install
```

Run the development server

```
npm start
```
